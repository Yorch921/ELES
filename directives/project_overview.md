# Proyecto ELES - Resumen General

ELES es una aplicación web de membresías diseñada para gestionar contenido exclusivo para clientes.

## Propósito
El sistema permite a los usuarios:
- Acceder a planes de entrenamiento y nutrición personalizados.
- Visualizar contenido educativo en formato video.
- Realizar pagos y suscripciones a través de una integración con PayPal.
- Gestionar su perfil y estado de membresía (Simple/Premium).

## Stack Tecnológico
- **Frontend**: React 19 (con Vite).
- **Enrutamiento**: React Router Dom.
- **Estado/Autenticación**: Context API nativo de React.
- **Pagos**: @paypal/react-paypal-js.
- **Estilos**: CSS modular nativo.

## Funcionalidades Actuales
- Landing page informativa.
- Sistema de Login (Frontend Mockup/Context).
- Panel de visualización de vídeos.
- Selector de planes y membresías.
